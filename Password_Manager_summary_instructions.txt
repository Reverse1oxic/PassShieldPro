This program allows you to store your passwords in an interactable .txt file. 
There are 3 main functions in this code add,delete, and view which all interact with the .txt file. 
This program also has a encryption and decryption function that allows the user to encrypt their file with a random secure code and decrypt with the same code.

**Be sure when using code you have a local file named passwords.txt for this code to run with no errors or you may change the variable name to match your file name
